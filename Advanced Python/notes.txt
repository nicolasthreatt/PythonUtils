Basic Collections
    - list
        + Mutable sequences of values
    - tuple
        + Fixed sequence of values
    - set
        + Sequence of distinct values
    - dictionary
        + Collection of key-value pairs

Advanced Collections
    - namedtuple
        + Tuple with named fields
    - OrderedDict, defaultdict
        + Dictionaries with special properties
    - Counter
        + Counts distinct values
    - deque
        + Double-ended list object

Python Logging
    - Captures and record events while app is running
    - Useful debugging feature
        + Not always practical to debug in real time
    - Events can be categorized for easier analysis
    - Provides transaction record of a program's execution
    - Highly customizable output
